## Project setup
```sh
yarn install
```

### Compiles and hot-reloads for development
```sh
yarn run serve
```
⚠️ If use env files, run with --mode dev ⚠️
```sh
yarn run serve --mode dev
```
### Compiles and minifies for production
```sh
yarn run build
```

### Run your tests
```sh
yarn run test
```

### Lints and fixes files
```sh
yarn run lint
```
